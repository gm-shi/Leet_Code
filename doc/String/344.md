# 344. Reverse String

![Example](../../assets/344.png)

**Solution:**

### Two Pointers

```java

class Solution {
    public void reverseString(char[] s) {
        int right = s.length - 1;
        for(int left = 0; left < right; left++) {
            char tmp = s[left];
            s[left] = s[right];
            s[right] = tmp;
            right--;
        }
    }
}

```
